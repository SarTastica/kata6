Kata 6: API REST & Backend
> **El reto:** *Convertir la app de escritorio en un servidor web.*

Eliminamos la interfaz gráfica antigua y desplegamos un **Servidor Web** con **SparkJava**.
* Los datos ahora se sirven como **JSON** usando la librería **Gson**.
* Aplicamos el **Patrón Adapter** (`TitleCommand`) para reutilizar toda la lógica de negocio de las katas anteriores y adaptarla al mundo web.
